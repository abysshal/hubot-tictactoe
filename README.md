# hubot-tictactoe

This is a TicTacToe game enigne for [Hubot](http://hubot.github.com/)

## Install

- git clone https://github.com/abysshal/hubot-tictactoe
- mkdir mybot
- cd mybot
- yo hubot
- npm link ../hubot-tictactoe
- add `hubot-tictactoe` to `external-scripts.json`


## Usage

- mybot ttt help
