# hubot-tictactoe

This is a TicTacToe game enigne for [Hubot](http://hubot.github.com/)

See `doc/*.md` for full documentation.

## Installation

- mkdir mybot
- cd mybot
- yo hubot
- npm install --save hubot-tictactoe
- add `hubot-tictactoe` to `external-scripts.json`


## Usage

- mybot ttt help
